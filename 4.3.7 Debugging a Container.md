### 7. Debugging Container 

### Development
Troubleshooting Docker Containers is different as we lose access to some of our development tools.

### Viewing Logs
`docker logs`

### Attaching to a Container
`docker exec -it sh`

### View Docker Processes
`docker ps`

### View Details of Docker Objects
`docker inspect`
